# 收件箱

这里是低摩擦捕捉区。任何来不及判断归属的闪念、问题、片段、链接或摘录都可以先放在这里。

## 使用方式

- 文件名建议为 `YYYY-MM-DD-简短描述.md`。
- 极短内容使用 [闪念模板](../模板/闪念.md)。
- 不必在记录当下完善分类、来源和链接。
- 收件箱不是永久住所；每周复盘时处理一次。

## 处理结果

每条内容最终应选择其一：

- 发展成 `10-思考` 中的思考卡片；
- 整理成 `20-百科` 中的稳定条目；
- 作为来源放入 `30-资料`；
- 进入一个分析或写作项目；
- 与已有笔记合并；
- 确认无价值后移入 `90-归档`。

处理完一条内容后，在目标笔记中保留必要信息，然后移动或归档原文件。
